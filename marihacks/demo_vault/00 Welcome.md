# Welcome to your Monika vault

This is a seed vault Monika will read from. She will not give you answers. She will quiz you on what you have written here.

## How it works

- Every note is chunked by heading and embedded into Monika's memory.
- When you ask a question, Monika pulls the relevant chunks and asks you about them.
- Edit any note and the constellation view will light up the memory galaxy as it reindexes.

## Try this first

1. Open the Monika chat pane from the left ribbon.
2. Ask something vague like "explain recursion for me".
3. Watch the reasoning validator score drop and Monika push back.
4. Try again with evidence of effort. Watch the effort score climb.
